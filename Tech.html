Tools and technologies

The tools required for this are Arduino nano microprocessor, 9V battery, temperature and humidity sensors, wires, bread board and LCD unit for display.

The team will also require a computer capable enough to run small-scale programming software such as the Arduino IDE where the code for the project will be written.

Finally, a guide to use these electrical components is needed, which can be found by using the internet and some knowledge of how to use the Arduino Ide which can also be learnt quickly using the internet.

All the components needed are relatively inexpensive and the software is free to download from the Arduino’s official website.

The chatbot prototype we experimented with was written in Python and utilised a module called chatterbot which incorporated natural language processing technologies into Python which could then be used.

In order for the program to run Python 3.x would need to be installed as well as the chatterbot and NLTK modules for the application to run correctly. All other modules are built-in to the Python interpreter and wouldn’t be required by the user. 

In terms of the GUI prototype we used the open source Tkinter module built for Python to create a simple interface compatible with Windows, macOS and Linux systems providing both dialog boxes and input boxes for users to enter their questions.
